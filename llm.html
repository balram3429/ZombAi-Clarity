👁 Self-Attention — How Tokens Talk to Each Other

Attention allows every token to "look at" every other token and decide how much to pay attention to it. The result: the model understands context and relationships across the entire sequence simultaneously.

Attention Matrix — darker = more attention
Attention(Q,K,V) = softmax( Q·Kᵀ / √dₖ ) · V
Q=Queries, K=Keys, V=Values — all learned linear projections
Q

Query

"What am I looking for?" — Each token asks a question about which other tokens are relevant to it.

K

Key

"What do I offer?" — Each token broadcasts what information it contains.

V

Value

"What do I give?" — The actual information passed along when attention is high.

🎯 RLHF — Making Models Actually Useful

Raw LLMs generate text, not helpful responses. Reinforcement Learning from Human Feedback (RLHF) aligns model outputs with human preferences — this is how ChatGPT, Claude, and Gemini are trained to be assistants.

RLHF Training Pipeline
1

Supervised Fine-tuning

Train on human-written demonstrations of ideal behavior.

2

Reward Model

Humans rank outputs; a reward model learns what "good" means.

3

PPO Training

Use RL to optimize the LLM to maximize the reward model's score.
